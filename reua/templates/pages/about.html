{% trans "Про фонд" %}

{% blocktrans %} Мета фонду — створити повноцінну інвестиційну платформу для відбудови України через будівництво нових сучасних кварталів, повернення українців додому та масштабування цієї моделі по всій країні. {% endblocktrans %}

{% blocktrans %} Фонд Rebuild Our Ukraine ініціює проєкт «Маріупольський квартал» як практичну модель післявоєнного відновлення: не тимчасове житло, не модульне містечко, а повноцінне міське середовище, де люди отримують дім, роботу, сервіси, освіту, медицину та можливість будувати майбутнє на рідній землі. {% endblocktrans %}

{% blocktrans %} Наша стратегічна ціль — залучити державних партнерів, міжнародних донорів, приватних інвесторів, громади та бізнес до створення нових житлово-інфраструктурних кварталів по всій території України. {% endblocktrans %}

{% blocktrans %} «Маріупольський квартал» — це пілотна модель, яку можна масштабувати в інших регіонах країни за участю всіх зацікавлених державних, громадських і приватних партнерів. {% endblocktrans %}

{% blocktrans %} Основний напрям діяльності фонду — створення системи децентралізованого водопостачання України, що забезпечує людей чистою питною водою. Для реалізації проєкту вибудовується довгострокова бізнес-модель із залученням інвесторів та міжнародних партнерів. {% endblocktrans %}

{% blocktrans %} Фонд створює не один житловий проєкт, а нову модель відновлення України. люди повертаються додому; держава отримує інструмент демографічного відновлення; інвестори отримують довгостроковий інфраструктурний актив; Україна отримує масштабований механізм відбудови. {% endblocktrans %}
